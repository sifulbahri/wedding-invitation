# 💍 Siful & UCU Yuli — Wedding Invitation Website

> **A digital wedding invitation crafted with love, engineered with passion.**

This project is more than a website.

It represents my personal journey toward one of the most important days of my life—my wedding with **Yulianti Agustina** on **15 November 2026**. As a cybersecurity professional and software engineer, I wanted to build something that wasn't just visually beautiful, but also reflected who I am: someone who enjoys creating meaningful technology.

Instead of relying on a template, I decided to build our invitation from scratch using modern web technologies, allowing every animation, interaction, and visual element to become part of our story.

---

# ❤️ Our Story

Every relationship has memories.

Some people preserve them in photo albums.

Some people write them in books.

I chose to preserve ours in code.

Every component in this project represents countless hours spent learning, designing, fixing bugs, improving animations, and polishing the smallest details so our family and friends can experience a memorable digital invitation.

This project reminds me that software isn't only for businesses or enterprises.

Sometimes software exists simply because someone loves another person.

# ✨ Project Goals

The primary objectives of this project are:

- Create a luxurious digital wedding invitation
- Deliver a cinematic user experience
- Ensure responsive design across devices
- Build reusable React components
- Practice modern frontend architecture
- Learn animation and UI composition
- Create a memorable experience for guests

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/<your-username>/wedding-invitation.git
```

## Install

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
